# Demo Rails App

#### This is a demo Ruby on Rails application with a ridiculous amount of tests (Code to Test Ratio: 1:4.1)

### Getting Started:
 - Clone & CD into the repo
 - `bundle install`
 - `docker-compose up -d`
 - `rails s -b 0.0.0.0`
 - [http://localhost:3000]([http://localhost:3000])

### Running Tests:
 - `rspec`

### View Stats:
 - `rails stats`
